# 🌍 ROAM - Tourist Spot Management Platform

ROAM is a full-stack MERN-based tourist spot management platform where travelers can explore destinations, add their own tourist spots, manage their contributions, and discover places shared by others.

## 🔗 Live Website

🌐 Live Site: https://tourist-fontend.vercel.app/

## 📂 GitHub Repositories

### Client Repository

https://github.com/moHasan9927/tourist-fontend

### Server Repository

https://github.com/moHasan9927/tourist-backend

---

## ✨ Features

- 🔐 Firebase Authentication (Email/Password & Google Login)
- 👤 User profile update (name and photo)
- ➕ Add new tourist spots
- 📖 View all tourist spots
- 🔍 View detailed information about each spot
- ✏️ Update tourist spots added by the user
- 🗑️ Delete tourist spots with confirmation
- 📋 My List page for managing personal spots
- 🌙 Dark / Light theme toggle
- 📱 Fully responsive design

## 🚀 Installation

### Clone Client

```bash
git clone https://github.com/moHasan9927/tourist-frontend.git
cd tourist-frontend
npm install
npm run dev
```

### Clone Server

```bash
git clone https://github.com/moHasan9927/tourist-backend
cd tourist-backend
npm install
npm start
```
